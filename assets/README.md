# bASICs Assets

Standalone v1-provided branding assets for the bASICs VM.

The VM should fetch this repository at a pinned commit instead of carrying
binary artwork inside the main NixOS configuration.

## Contents

- `assets/wallpapers/`: v1 wallpaper PNG/WebP assets.
- `assets/logos/`: v1 logo PNG/WebP assets.
- `assets/faces/`: v1 account face image.
- `nix/package.nix`: simple fixed-output install derivation for use from the VM repo.
