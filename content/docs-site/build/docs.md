# Work on the Docs

The docs source is in `content/docs-site`.

## Get the source

```bash
git clone https://github.com/ZimengXiong/basicsvm.git
cd basicsvm
```

## Preview

```bash
cd content/docs-site
npm install
npm run dev
```

Open:

```text
http://localhost:5173/
```

## Build

```bash
cd content/docs-site
npm run build
```
