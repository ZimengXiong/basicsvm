# Work on the Docs

The docs source is in `content/docs-site`.

## Preview

```bash
cd content/docs-site
npm install
npm run dev
```

Open:

```text
http://localhost:5173/
```

## Build

```bash
cd content/docs-site
npm run build
```

## VM docs launcher

The VM does not include a separate offline docs build. The desktop docs launcher opens:

```text
https://basics.alpacawebservices.com
```
