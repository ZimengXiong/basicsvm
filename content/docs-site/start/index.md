# Start Here

Pick the computer you are using, then follow that page. Each page has the VM download, the app download, and the minimum system requirements.

| Your computer | Use this page |
| --- | --- |
| Mac with Apple Silicon, such as M1, M2, M3, or newer | [macOS Apple Silicon](../install/mac-apple-silicon.md) |
| Mac with an Intel processor | [macOS Intel](../install/mac-intel.md) |
| Windows PC with Intel or AMD | [Windows x86](../install/windows-x86.md) |
| Windows PC with ARM | [Windows ARM](../install/windows-arm.md) |
| Linux PC with Intel or AMD | [Linux x86](../install/linux-x86.md) |

After installation, continue to [First Boot](./first-boot.md), then run [First Flow](../use/first-flow.md).
