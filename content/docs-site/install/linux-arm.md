# Linux ARM

Use this page for Linux machines with an ARM processor.

> [!WARNING]
> Make sure your computer has at least 30 GB of free storage for the VM download and import.

## Install

1. Download the bASICs VM for Linux ARM. <!-- TODO: add direct download link -->
2. Create an ARM virtual machine with QEMU.
3. Attach the downloaded `.qcow2` as the VM disk.
4. Forward VNC or another display connection from the guest.
5. Start the VM and connect to the display.

Linux ARM setup depends on your VM tooling. Use QEMU or another ARM-capable setup and connect over VNC or an equivalent display path.

The desktop logs in automatically.
