# Linux x86

Use this page for Linux PCs with an Intel or AMD processor.

> [!WARNING]
> Make sure your computer has at least 30 GB of free storage for the VM download and import.

## Install

1. Install [VirtualBox for Linux](https://www.virtualbox.org/wiki/Linux_Downloads).
2. Download [bASICs VM for Linux x86](https://huggingface.co/datasets/zimengxiong/basicsvm/resolve/main/releases/55ea8cb/linux-x86/basicsvm-x86_64-linux.ova?download=true).
3. Open VirtualBox.
4. Choose **File > Import Appliance**.
5. Select the downloaded `.ova`.
6. Start the VM.

The desktop logs in automatically. For SSH or terminal login:

```text
user: beaver
password: works
```

Next: [First Boot](../start/first-boot.md).
